# Changelog

Todas las novedades y mejoras implementadas en **Arbora**.

## [1.1.0] - 2026-03-29

### Añadido
-   **Botón de Copiar ID**: Se agregó un botón rápido en la tabla de mensajes del Dashboard para copiar el ID (número de teléfono) de los usuarios con un solo clic.
-   **Feedback Visual de Copiado**: Animación y cambio de color al copiar exitosamente un ID al portapapeles.

### Mejoras
-   **Optimización de UI**: Ajustes menores en el espaciado de la tabla de mensajes para una mejor legibilidad.
-   **Documentación**: Actualización del README y creación de este archivo CHANGELOG para un mejor seguimiento del proyecto.

---

## [1.0.0] - 2026-03-27

### Añadido
-   **Lanzamiento Inicial**: Versión base con Dashboard, Editor de Flujos Visual y soporte para WhatsApp/Telegram.
-   **Editor de Flujos**: Implementación de `@xyflow/react` para la creación visual de bots.
-   **Analíticas**: Gráficos dinámicos con Chart.js.
-   **Gestión de Sesiones**: Login/Registro y soporte para múltiples instancias de WhatsApp.
